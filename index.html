01

A Redis-backed DAG scheduler for heterogeneous ML pipelines.

PCB Auto · AI Infrastructure Intern · summer 2025

The existing training pipeline was a naive FIFO queue — workflows blocked on each other, and a third of the GPU fleet sat idle. I replaced it with a Redis-backed distributed scheduler with DAG dependency resolution, resource-aware tag streams, and topological execution across A100 / L40S / L4 nodes.

For reliability I designed the fault-tolerance layer: TTL worker heartbeats, idempotent job keys (hashing DAG ID + input checksum), exponential backoff with at-least-once delivery, and an observability stack that surfaces per-stage latencies and stragglers.

02

Two co-located perception models on one Jetson Orin AGX.

X-Lab, SUNY Buffalo · Visual Perception Research Engineer · 2022 — 2023

Deploying a multi-view BEV perception model alongside a 3D occupancy network on a single embedded GPU meant fighting memory bandwidth, kernel-launch overhead, and stream contention simultaneously. I owned the full optimization path: custom TensorRT plugins, INT8 quantization, and Triton ensemble pipelining across dedicated CUDA streams for the 6-camera BEV model.

For the coexisting occupancy network, a naive NCDHW layout wrecked DRAM coalescing. I authored an ONNX layout rewriter (NCDHW → NDHWC), added layer-wise mixed precision, and capped mIoU loss at 0.38%. CUDA Graph capture, in-place voxel tensor recycling, stream-priority coordination, and a custom warp-level argmax kernel made both models coexist within the memory and stream budget.

03

An edge-cloud split for end-to-end autonomous driving.

Rochester Institute of Technology · research

End-to-end driving stacks that want both CNN perception and LLM reasoning blow past Orin AGX's power envelope. I architected a hybrid serving framework that shards heavy CNN perception (UniAD) onto the edge via quantized TensorRT engines, and offloads the LLM reasoning head (DriveVLM) to the cloud through vLLM inside Triton — with PagedAttention and continuous batching to pay for the round-trip.

A 1 kHz IMU fusion + ONNX plugin for VR haptics.
